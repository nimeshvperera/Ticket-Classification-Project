# Ticket-Classification-Project

🎫 AI Ticket Classification

Goal

Built an NLP-based system to automatically classify support tickets, reducing manual triage time and improving routing efficiency.

Tech Stack

Python, scikit-learn, pandas, NumPy, TF-IDF, NLP preprocessing, Logistic Regression / Decision Trees

Challenges & Solutions

	•	Messy, unstructured text → Cleaned and normalized data (tokenization, stopword removal)
	•	Imbalanced classes → Applied resampling techniques to improve model fairness
	•	Model accuracy tuning → Compared multiple models and optimized with feature engineering (TF-IDF)
	•	Generalization → Validated on unseen data to avoid overfitting

Impact

Delivered a scalable ML pipeline that can classify incoming tickets in real-time, reducing manual workload and enabling faster response times.
